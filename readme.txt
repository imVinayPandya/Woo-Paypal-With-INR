This plugin enable inr for paypal gateway. its automatically convert inr to usd when customer try to check out with paypal.

== Description ==

This plugin enable Paypal Payment Gateway for INR currency in Woocommerce Plugin. and its allow you too select INR as base currency and when customer going to use paypal as payment method, this plugin automatically convert INR to USD because Paypal not accept INR currency its only allow you to USD. this way you can use INR with Paypal.

Conversion rate of INR to USD it will select from yahoo api automatically, so need to worry about conversion rate.


== Installation ==

1. Upload `WooCommerce INR for Paypal ` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy INR for you woocommerce store


after activating this plugin you do not need to do anything. and you should already installed woocommerce plugin before installing this plugin.

== Frequently asked questions ==

= A question that someone might have =

1) how is works?
ans: Paypal does not support INR but its support USD. So This plugin will convert your INR to USD using yahoo market api, before customer reach to paypal gateway. and this way you can enable INR for Paypal.

2) how its get currency conversion rate of INR ?
ans: we are converting INR to USD from yahoo market api.
it will take current rate of USD from yahoo market api. so you no need to worry about if USD rate is change for INR, it will get latest exchange ratee for INR.
